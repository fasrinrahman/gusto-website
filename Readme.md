# 🍽️ Gusto Restaurant Website

A modern, responsive restaurant website built using **HTML5 and CSS3**.  
This project showcases a complete restaurant landing page with menu, gallery, and reservation form.

---

## 📌 Project Overview

**Gusto Restaurant Website** is a static frontend project designed to present a restaurant’s:

- Brand identity  
- Food specials  
- Menu offerings  
- Chef introduction  
- Reservation system  

The design focuses on **clean UI, responsiveness, and user experience**.

---

## 🧠 Key Features

✅ Fully responsive design (Desktop, Tablet, Mobile)  
✅ Modern layout using CSS Grid & Flexbox  
✅ Hero section with navigation bar  
✅ Specials section with featured dishes  
✅ Restaurant story section  
✅ Menu categorized (Breakfast, Main, Desserts, Drinks)  
✅ Image gallery  
✅ Chef introduction section  
✅ Reservation form UI  
✅ Footer with contact details  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- Google Fonts (Playfair Display, Raleway)
- Responsive Design (Media Queries)
